VitePress 文章底部时间 ​

这篇文章是VuePress上 last updated 最后更新时间的设置，可能会和VitePress有些许不同。内容并不是如官网给一个这个配置而打开最后更新时间的选项：

ts
export default defineConfig {
  themeConfig: {
    lastUpdated: true
  }

  // or open it and set last updated text
  themeConfig: {
    lastUpdated: {
      text: '最后更新于: '
    }
  }
}
